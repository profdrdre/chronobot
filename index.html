The Chronobot Helper

This is an unofficial implementation for the solo-mode bot for the board game anachrony published by Mindclash Games. In the solo-mode an algorithm called the Chronobot takes the action of the competing player. The official rules can be found here. The Chronobot helper allows you to play the Chronobot without the need of setting up its board. It also does a lot of the bookkeeping.
